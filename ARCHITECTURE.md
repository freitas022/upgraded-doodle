# Marketplace Backend

Tecnologias:

- Java 21
- Spring Boot 4
- PostgreSQL
- Lombok
- Gradle

Arquitetura:

- Domain
- Infrastructure
- Presentation

Regras obrigatórias:

- Aplicar SOLID
- Utilizar Records para DTOs
- Utilizar Constructor Injection
- Não utilizar Field Injection
- Não criar lógica de negócio em Controllers
- Regras de negócio devem ficar em Services

Antes de gerar código:

1. Analise entidades existentes
2. Reutilize componentes existentes
3. Evite duplicação
4. Mantenha consistência arquitetural
